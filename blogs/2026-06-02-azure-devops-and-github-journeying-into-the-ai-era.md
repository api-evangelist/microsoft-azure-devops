Microsoft is advancing AI-powered development by prioritizing GitHub for new agentic capabilities while supporting hybrid workflows with Azure DevOps, including the preview of Enterprise Live Migrations for moving repositories to GitHub with minimal downtime. New Azure DevOps features include Copilot Code Review, Autofix for CodeQL security alerts, and Apple Silicon Mac support for Azure Pipelines with pay-per-minute billing. The announcement signals Microsoft's strategic direction of using GitHub as the primary surface for AI-native developer tooling while Azure DevOps continues serving project management and CI/CD needs.
