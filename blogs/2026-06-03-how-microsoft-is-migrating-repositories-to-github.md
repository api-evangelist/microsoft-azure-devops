Microsoft's Copilot, Agents, and Platforms organization has migrated over 1,600 repositories and 3,100 developers from Azure DevOps to GitHub in six months, leveraging migration tools including GitHub Enterprise Importer and Enterprise Live Migrator. The company maintains a hybrid approach, keeping Azure Boards and Azure Pipelines for project management while using GitHub repositories to access AI-native development capabilities including GitHub Copilot Coding Agent and agentic workflows.
